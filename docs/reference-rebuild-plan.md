# Reference-Inspired Rebuild Plan

참고 사이트 `https://bbbang105.github.io/`는 포트폴리오와 블로그가 결합된 구조입니다.
이 프로젝트에서는 디자인을 그대로 복제하지 않고, 아래 요소를 참고해 재구성합니다.

## 참고할 핵심 구조

1. 좌측 고정형 소개/내비게이션 영역
2. 메인 영역의 이력서형 섹션 구성
3. 최근 게시글 요약과 섹션 목차 공존
4. 검색, 언어, 다크 모드처럼 확장 가능한 유틸 포인트

## 재구성 단계

1. 홈 구조 재편
   - 학생/개발자 소개
   - 섹션형 포트폴리오 레이아웃
   - 최근 게시글/콘텐츠 스냅샷 정리

2. 콘텐츠 탐색 구조 확장
   - 카테고리/태그 탐색 강화
   - 홈 섹션과 블로그/게시판 연결
   - 섹션 목차 링크 정리

3. 작성 경험 보강
   - Markdown 본문 작성
   - 미리보기
   - 코드 블록과 이미지 본문 삽입

4. 운영 기능 강화
   - 검색
   - 고정글
   - 페이지네이션
   - 조회수와 공개 상태 관리

5. 스토리지/배포 준비
   - 로컬 첨부파일 구조를 Supabase Storage로 전환
   - 이후 Ubuntu + PM2 + Nginx 배포
